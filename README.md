# Task Bloom - To-Do List App

## Project Overview
Task Bloom is a client-side JavaScript to-do list web application created for COMP1073 Assignment 3. The app allows users to add, complete, filter, and delete tasks through a simple and visually appealing interface.

This project was built using HTML, CSS, and JavaScript code.

## Features
- Add new tasks
- Mark tasks as completed using a checkbox
- Move completed tasks to the bottom of the list
- Apply a line-through style to completed tasks
- Delete individual tasks
- Clear all completed tasks
- Clear all tasks
- Filter tasks by:
  - All
  - Active
  - Completed
- Display live task statistics:
  - Total tasks
  - Completed tasks
  - Remaining tasks
- Save tasks using local storage so they remain after refreshing the page
- Optional completion sound effect
- Responsive layout for smaller screens
- Animated task deletion effect

## Technologies Used
- HTML5
- CSS3
- JavaScript
